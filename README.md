# Micro-Shop

Team:
* David Hofstetter
* Justin Calle
* Sananjayan Paramanantharajah


Unser Frontend ist mit einem Backend verknüpft, welches die Anfrage an dementsprechenden Backend schickt und die Antwort zurückleitet.
Unsere Services sind:
* Gateway
* Produktkatalog
* Kundenverwaltung
* Zahlungsservice
* Warenkorb
* Bestellungsservice
